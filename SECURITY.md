# Security Policy

SmartQueue AI aims to follow production-grade security practices.

## Supported versions

Only the latest released version is supported.

## Reporting vulnerabilities

If you discover a security issue, please report it privately to the project
maintainers through the repository security contact or your preferred private
channel.

Please include:

- A concise description of the issue
- The affected endpoint, page, or workflow
- Steps to reproduce
- Any relevant logs or request/response examples

Do not open public issues for active vulnerabilities.
